Ideas principales


Texto en html

Para el texto en HTML se puede usar de distintas formas como lo que son los encabezados, párrafos, negrita,cursiva , listado ordenado y desordenado, espacios, saltos de líneas y comentarios,
Esto nos sirve para darle una estructura, una mejor legibilidad y accesibilidad a nuestra pagina.

imágenes en HTML y CSS

Las imágenes tienen una gran importancia ya que ayudan al usuario a comprender el contenido de la página y haciendo mucho mas presentable la pagina o poniendo de fondo una imagen para darle mas personalidad a la pagina.

Enlaces en HTML

Los enlaces nos sirven para poder crear vínculos a otras páginas, archivos, secciones dentro de la misma página.

Proceso de diseño

Planificación: Definir la estructura y objetivos del sitio.
HTML: Crear la estructura básica de la página.
CSS: Aplicar el estilo y diseño visual.
Diseño responsivo: Asegurar que el sitio funcione en todos los dispositivos.
Pruebas: Verificar la funcionalidad y apariencia en diferentes plataformas.
Optimización: Mejorar la velocidad y eficiencia del sitio antes de publicarlo.

En resumen es muy necesario y indispensable para poder hacer una pagina web ya que con todo lo mencionado sirve para dar estructura, estilo, y orden

Elemenetos


Texto HTML

Se usa < p> para parrafos
< h1> Para titulos
< h2> Para subtitulos
< h3>
< h4> Y asi susesivamente.
< strong> Para las letras en negritas.
< em> Para letras cursivas.
< ol> Para lista Ordenada.
< ul> para lista ordenada.
< br> Para un salto de linea.
< q> Para citas en linea.
< blockquote> citas largas
< !-- comentario -->. para agrergar comentario.
< img src="" alt=""> para agregar una imagen.
< a> para etiquetas.

Resumen

En resumen, es importante conocer cada uno de estos elementos mencionados, ya que, en mi opinión, es la base para poder empezar a escribir un código en HTML. Es necesario tener orden, coherencia y creatividad, pero de una forma que se pueda plasmar en el HTML o, mejor dicho, en la página web
